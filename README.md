# MGA Site

Website institucional estático da **Miguel, Goes & Associados – Sociedade de Advogados, RL**, desenvolvido em HTML, CSS e JavaScript vanilla.

## Estado atual

O projeto encontra-se funcional e editorialmente consistente, com as principais páginas institucionais alinhadas numa apresentação sóbria e profissional:

- `index.html` — homepage institucional
- `sobre.html` — apresentação da sociedade
- `equipa.html` — apresentação da equipa
- `areas.html` — áreas de atuação
- `contactos.html` — contactos, formulário e mapa
- `politica-privacidade.html` — política de privacidade
- `informacao-legal.html` — informação legal

O site usa:

- `css/estilos.css` — estilos globais
- `js/script.js` — interações comuns (fade-in, hero, menu mobile, scroll reveal, header e footer bar)
